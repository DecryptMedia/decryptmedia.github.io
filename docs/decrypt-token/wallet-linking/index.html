Wallet Linking

The token migration dapp is live at decrypt.co/token-status. You must first log in to the Decrypt.co website with the same email you used to create your account in the Decrypt mobile app.

If you are logging in to the Decrypt website on your mobile device, you will need to use the Metamask app and browser to link a wallet.

Please follow the step in the video for the platform you are linking your wallet on.

Read more about the token migration in the Migration FAQ.
